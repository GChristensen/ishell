iShell Extension

Install Firefox Add-On

SEE ALSO: Enso open-source, Scrapyard

Description

iShell is a browser add-on available by Ctrl+Space key combination that lets you give commands to a browser by entering text into a special input box. Commands are shortcuts for things that you commonly do on the Web; they can eliminate repetitive tasks and let you get more done, faster. Your can create your own iShell commands in JavaScript.

Find more information on command authoring in the tutorial at iShell setting pages.

History

iShell is a descendant of the Mozilla Ubiquity project that was phased out with the advent of Firefox Quantum and its restricting Web Extensions.

iShell is aimed to bring back the full functionality of Ubiquity to the new Firefox and enhance it further. iShell contains the advanced Ubiquity command parser ported to modern ECMAScript, which allows to create commands with prepositional arguments. It also implements the large portion of the original Ubiquity API that helps to migrate existing legacy Ubiquity commands to Firefox Quantum almost seamlessly. But the main point of iShell creation is to provide the new clean object-oriented command authoring syntax based on the modern language features.

Although there is no more command subscription model for security reasons (you need to manually maintain third-party commands in the iShell command editor), iShell is still immensely useful if you are able to write JavaScript to automate things you have (web)APIs for or to extend the browser without much effort.

Some additional features not found in the original Ubiquity

  • More argument roles and prepositions: 'by' (cause), 'for' (subject) and 'on' (dependency).
  • Ability to add commands to the browser context menu.
  • It is possible to separate user commands to multiple categories in the command editor.
  • Modern object-oriented command syntax.

Create your own commands

iShell allows to declaratively define and automatically parse commands with prepositional arguments. For example, consider the command show-text with the following syntax: "show-text text in popup". It has the prepositional argument named in and an implicit nameless first argument. iShell command parser will automatically provide values (text, popup) of the arguments to command implementation handler methods after you enter the command in the input box.

iShell commands also have full access to jQuery and browser WebExtension APIs, which give you ability to extend the browser almost effortlessly. See an example.

Known issues

Some builtin commands now require API keys of their corresponding services to work. If the help links at the iShell interface provide not enough guidance to obtain them, please see project GitHub page.
