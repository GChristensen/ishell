You Have commands in categories. | Show Help

  • All the commands that iShell recognizes are listed in the right column. The black text is the command's name (what you type into iShell to use it). The grey text explains what the command does.
  • The left column shows all the command categories available.
  • If there's a command you don't want, uncheck the box next to it. You can get the command back by re-checking the box.
