  • Add new dynamic settings with add-setting setting_value as setting_key command.
  • All the dynamic settings are available through cmdAPI.settings property in the code.
  • Some settings are used by the corresponding builtin commands which behavior you can customize.
  • To use Google commands you need to create a custom search engine with two enabled options: "Search the entire web" and "Image search".
